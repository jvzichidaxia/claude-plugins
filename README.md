# Claude Plugins

Personal Claude Code plugin marketplace.

## Available Plugins

| Plugin | Description | Install |
|--------|-------------|---------|
| **symbiosis** | Cognitive symbiosis system — build a living AI model of yourself | `claude plugin install symbiosis` |
| **journal** | AI-powered daily journal from Claude Code conversations | `claude plugin install journal` |

## Setup

```bash
# Add this marketplace (one-time)
claude plugin marketplace add jvzichidaxia/claude-plugins

# Install any plugin
claude plugin install symbiosis
claude plugin install journal
```
